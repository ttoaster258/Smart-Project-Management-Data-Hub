# 智能报告生成系统技术方案

## 一、方案概述

### 1.1 业务背景
当前智慧项目管理数据中心已积累了丰富的项目数据（财务指标、里程碑进度、变更记录、风险预警等），管理层每周/月/年需要手动汇总这些数据撰写报告，耗时费力且容易遗漏关键信息。本方案旨在构建一套**智能报告自动生成系统**，通过 AI 大模型自动从数据库提取关键指标并生成专业报告。

### 1.2 核心价值
| 价值维度 | 具体收益 |
|---------|---------|
| **效率提升** | 报告撰写时间从 2-4 小时缩短至 5-10 分钟 |
| **数据准确性** | 自动提取数据库真实数据，避免人工统计误差 |
| **洞察深度** | AI 自动识别异常数据、趋势变化、潜在风险 |
| **格式灵活** | 支持 Word、PDF、HTML 多格式输出 |
| **可追溯** | 每份报告记录生成日志，支持版本对比 |

### 1.3 系统定位
本系统作为现有平台的功能扩展，遵循现有技术栈（Express + SQLite），新增独立的报告生成模块，不影响现有功能。

### 2.2 技术选型

| 组件 | 推荐方案 | 选型理由 |
|-----|---------|---------|
| **AI 模型** | Claude API (Anthropic) | 中文能力强、上下文窗口大(200K)、输出格式可控、API 稳定 |
| **报告渲染** | docx + pdfkit | 支持 Word/PDF 双格式，纯 Node.js 实现 |

| **历史存储** | SQLite 新增表 | 统一存储，便于查询追溯 |

### 3.2 数据提取 SQL 示例

```sql
-- 周报数据提取核心 SQL
-- 项目统计概览
SELECT 
  COUNT(*) as total_projects,
  SUM(CASE WHEN kickoff_date BETWEEN :startDate AND :endDate THEN 1 ELSE 0 END) as new_projects,
  SUM(CASE WHEN status = '已验收' AND acceptance_date BETWEEN :startDate AND :endDate THEN 1 ELSE 0 END) as completed,
  SUM(CASE WHEN status = '正在进行' THEN 1 ELSE 0 END) as ongoing,
  SUM(CASE WHEN status = '延期' THEN 1 ELSE 0 END) as delayed,
  SUM(contract_amount) as total_contract,
  SUM(annual_confirmed_revenue) as total_revenue,
  SUM(paid_2026) as total_payment
FROM projects;

-- 区域业绩明细
SELECT 
  region,
  COUNT(*) as project_count,
  SUM(contract_amount) as contract_amount,
  SUM(annual_confirmed_revenue) as revenue,
  SUM(paid_2026) as payment
FROM projects
GROUP BY region
ORDER BY revenue DESC;

-- 变更统计
SELECT 
  type,
  COUNT(*) as change_count,
  GROUP_CONCAT(reason_category) as reasons
FROM project_changes
WHERE change_date BETWEEN :startDate AND :endDate
GROUP BY type;
```

## 四、AI 集成方案

### 4.1 API 接入方式

```javascript
// server/services/AIService.js
import Anthropic from '@anthropic-ai/sdk';

class AIService {
  constructor() {
    this.client = new Anthropic({
      apiKey: process.env.ANTHROPIC_API_KEY
    });
  }

  /**
   * 生成报告正文
   * @param {ReportContext} context - 结构化数据上下文
   * @param {string} reportType - 报告类型 (weekly/monthly/annual)
   * @param {string} template - 报告模板风格
   */
  async generateReport(context, reportType, template = 'professional') {
    const prompt = this.buildPrompt(context, reportType, template);
    
    const response = await this.client.messages.create({
      model: 'claude-sonnet-4-6',
      max_tokens: 8000,
      temperature: 0.3,  // 低温度保证输出稳定
      messages: [{
        role: 'user',
        content: prompt
      }]
    });

    return response.content[0].text;
  }

  /**
   * 构建提示词
   */
  buildPrompt(context, reportType, template) {
    const templateConfig = PROMPT_TEMPLATES[template];
    const typeConfig = REPORT_TYPE_CONFIGS[reportType];
    
    return `
${templateConfig.systemPrompt}

## 报告类型：${typeConfig.title}
## 时间范围：${context.timeRange.startDate} 至 ${context.timeRange.endDate}

## 数据概览
请基于以下真实数据撰写报告，所有数值必须严格使用提供的数据：

### 项目统计
- 项目总数：${context.overview.totalProjects} 个
- 本期新增：${context.overview.newProjects} 个
- 本期完成：${context.overview.completedProjects} 个
- 进行中：${context.overview.ongoingProjects} 个
- 延期项目：${context.overview.delayedProjects} 个

### 财务指标
- 合同总额：${formatCurrency(context.overview.totalContractAmount)}
- 确认收入：${formatCurrency(context.overview.totalRevenue)}
- 回款总额：${formatCurrency(context.overview.totalPayment)}
- 平均毛利率：${context.financialHealth.averageMarginRate.toFixed(2)}%

### 区域业绩排名
${context.regionPerformance.map(r => 
  `- ${r.region}：收入 ${formatCurrency(r.revenue)}，KPI达成率 ${r.kpiAchievement.toFixed(1)}%`
).join('\n')}

### 风险预警
- 进度风险项目：${context.riskAlerts.progressRisks.length} 个
- 成本风险项目：${context.riskAlerts.costRisks.length} 个
- 质量风险项目：${context.riskAlerts.qualityRisks.length} 个

### 环比对比
- 项目数环比增长：${context.comparison.growthRate.projectCount.toFixed(1)}%
- 收入环比增长：${context.comparison.growthRate.revenue.toFixed(1)}%
- 回款环比增长：${context.comparison.growthRate.payment.toFixed(1)}%

## 输出要求
${typeConfig.outputRequirements}

请生成完整的报告内容，格式使用 Markdown。
`;
  }
}
```

## 五、API 接口设计

### 5.1 RESTful API 规范

```javascript
// server/routes/reports.js

import express from 'express';
import { authenticate } from '../middleware/auth.js';
import ReportService from '../services/ReportService.js';

const router = express.Router();

/**
 * POST /api/reports/generate
 * 生成报告
 * 
 * Request Body:
 * {
 *   "type": "weekly" | "monthly" | "annual",
 *   "startDate": "2026-04-01",
 *   "endDate": "2026-04-07",
 *   "template": "professional" | "narrative" | "executive",
 *   "format": "markdown" | "html" | "docx" | "pdf",
 *   "includeCharts": true,
 *   "customSections": ["risk", "milestone"] // 可选，指定生成部分章节
 * }
 * 
 * Response:
 * {
 *   "success": true,
 *   "data": {
 *     "reportId": "RPT-20260416-001",
 *     "content": "...", // Markdown 内容
 *     "downloadUrl": "/api/reports/RPT-20260416-001/download",
 *     "generatedAt": "2026-04-16T10:30:00Z"
 *   }
 * }
 */
router.post('/generate', authenticate, async (req, res) => {
  try {
    const { type, startDate, endDate, template, format, includeCharts, customSections } = req.body;
    
    // 参数校验
    if (!type || !startDate || !endDate) {
      return res.status(400).json({
        success: false,
        error: '缺少必要参数：type, startDate, endDate'
      });
    }

    // 生成报告
    const result = await ReportService.generateReport({
      type,
      startDate,
      endDate,
      template: template || 'professional',
      format: format || 'markdown',
      includeCharts: includeCharts || true,
      customSections: customSections || null,
      generatedBy: req.user.username
    });

    res.json({
      success: true,
      data: result
    });
  } catch (error) {
    console.error('报告生成失败:', error);
    res.status(500).json({
      success: false,
      error: error.message
    });
  }
});

/**
 * GET /api/reports/history
 * 获取报告历史记录
 * 
 * Query Params:
 *   - type: 报告类型筛选
 *   - startDate: 开始日期筛选
 *   - endDate: 结束日期筛选
 *   - limit: 返回数量限制
 */
router.get('/history', authenticate, async (req, res) => {
  try {
    const { type, startDate, endDate, limit = 20 } = req.query;
    const history = await ReportService.getReportHistory({
      type,
      startDate,
      endDate,
      limit: parseInt(limit)
    });
    res.json({ success: true, data: history });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

/**
 * GET /api/reports/:id/download
 * 下载报告文件
 */
router.get('/:id/download', authenticate, async (req, res) => {
  try {
    const { format = 'docx' } = req.query;
    const filePath = await ReportService.getReportFile(req.params.id, format);
    res.download(filePath);
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

/**
 * POST /api/reports/regenerate/:id
 * 重新生成报告（使用相同参数）
 */
router.post('/regenerate/:id', authenticate, async (req, res) => {
  try {
    const result = await ReportService.regenerateReport(req.params.id);
    res.json({ success: true, data: result });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

/**
 * GET /api/reports/preview
 * 预览报告数据（不调用 AI，仅返回提取的数据）
 */
router.get('/preview', authenticate, async (req, res) => {
  try {
    const { type, startDate, endDate } = req.query;
    const data = await ReportService.extractReportData(type, startDate, endDate);
    res.json({ success: true, data });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

export default router;
```

```javascript
// server/services/DocxGenerator.js
import { Document, Packer, Paragraph, TextRun, HeadingLevel, Table } from 'docx';

class DocxGenerator {
  async generate(markdownContent, charts = []) {
    // Markdown 转 Word 结构
    const sections = this.parseMarkdownToSections(markdownContent);
    
    const doc = new Document({
      sections: [{
        properties: {},
        children: [
          // 标题
          new Paragraph({
            text: '项目管理报告',
            heading: HeadingLevel.TITLE,
            style: 'title'
          }),
          // 日期
          new Paragraph({
            text: `生成时间：${new Date().toLocaleString('zh-CN')}`,
            style: 'subtitle'
          }),
          // 正文内容
          ...sections.map(section => this.createParagraph(section)),
          // 图表（可选）
          ...(charts.length > 0 ? this.createChartSections(charts) : [])
        ]
      }]
    });

    return Packer.toBuffer(doc);
  }
}
```

## 七、微调方案（可选进阶）

### 7.1 是否需要微调？

**结论：初期不建议微调，优先验证效果**

| 场景 | 方案 | 理由 |
|-----|-----|------|
| **初期验证** | Prompt Engineering | Claude 中文能力足够，好的提示词可覆盖90%需求 |
| **中期优化** | Few-shot Learning | 收集10-20份优秀报告作为示例，提升风格一致性 |
| **长期定制** | 微调（如需） | 仅当报告风格极度特殊、需严格遵循企业格式时考虑 |

### 7.3 Skills 的使用

关于您提到的 Skills，这里有两种理解：

**理解一：AI Agent 工具调用（Tool Use）**
- Claude 支持通过 tool use 调用外部工具
- 本方案中已设计数据提取 API 作为工具
- AI 可以主动调用 `/api/reports/preview` 获取实时数据
- **推荐使用**：增强报告时效性和准确性

**理解二：企业定制技能模块**
- 如果指企业专属的"技能包"（如特定分析模板）
- 可以通过 JSON 配置文件实现
- 不需要模型微调，只需扩展 promptTemplates.js
- **推荐使用**：灵活可配置，维护成本低

## 八、前端界面设计

### 8.1 报告生成页面组件

```tsx
// components/ReportGeneratorPage.tsx

interface ReportGeneratorProps {
  onSubmit: (params: ReportParams) => void;
}

const ReportGeneratorPage: React.FC<ReportGeneratorProps> = () => {
  const [reportType, setReportType] = useState<'weekly' | 'monthly' | 'annual'>('weekly');
  const [dateRange, setDateRange] = useState({ start: '', end: '' });
  const [template, setTemplate] = useState('professional');
  const [format, setFormat] = useState('markdown');
  const [isGenerating, setIsGenerating] = useState(false);
  const [previewData, setPreviewData] = useState(null);

  // 预览数据
  const handlePreview = async () => {
    const response = await fetch(`/api/reports/preview?type=${reportType}&startDate=${dateRange.start}&endDate=${dateRange.end}`);
    const data = await response.json();
    setPreviewData(data);
  };

  // 生成报告
  const handleGenerate = async () => {
    setIsGenerating(true);
    try {
      const response = await fetch('/api/reports/generate', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          type: reportType,
          startDate: dateRange.start,
          endDate: dateRange.end,
          template,
          format
        })
      });
      const result = await response.json();
      // 显示报告或下载
      if (result.success) {
        window.open(result.data.downloadUrl);
      }
    } finally {
      setIsGenerating(false);
    }
  };

  return (
    <div className="p-6 bg-white rounded-xl">
      <h2 className="text-xl font-bold mb-4">智能报告生成</h2>
      
      {/* 报告类型选择 */}
      <div className="flex gap-4 mb-4">
        {['weekly', 'monthly', 'annual'].map(type => (
          <button
            key={type}
            className={`px-4 py-2 rounded ${reportType === type ? 'bg-blue-500 text-white' : 'bg-gray-100'}`}
            onClick={() => setReportType(type as any)}
          >
            {type === 'weekly' ? '周报' : type === 'monthly' ? '月报' : '年报'}
          </button>
        ))}
      </div>

      {/* 日期范围 */}
      <div className="flex gap-4 mb-4">
        <input type="date" value={dateRange.start} onChange={e => setDateRange({ ...dateRange, start: e.target.value })} />
        <input type="date" value={dateRange.end} onChange={e => setDateRange({ ...dateRange, end: e.target.value })} />
      </div>

      {/* 模板风格 */}
      <select value={template} onChange={e => setTemplate(e.target.value)} className="mb-4">
        <option value="professional">专业商务风格</option>
        <option value="narrative">叙述性风格</option>
        <option value="executive">执行摘要风格</option>
      </select>

      {/* 输出格式 */}
      <select value={format} onChange={e => setFormat(e.target.value)} className="mb-4">
        <option value="markdown">Markdown</option>
        <option value="docx">Word 文档</option>
        <option value="pdf">PDF</option>
      </select>

      {/* 操作按钮 */}
      <div className="flex gap-4">
        <button onClick={handlePreview} className="px-4 py-2 bg-gray-200 rounded">预览数据</button>
        <button onClick={handleGenerate} disabled={isGenerating} className="px-4 py-2 bg-blue-500 text-white rounded">
          {isGenerating ? '生成中...' : '生成报告'}
        </button>
      </div>

      {/* 数据预览 */}
      {previewData && (
        <div className="mt-4 p-4 bg-gray-50 rounded">
          <h3 className="font-bold mb-2">数据预览</h3>
          <pre>{JSON.stringify(previewData, null, 2)}</pre>
        </div>
      )}
    </div>
  );
};
```

## 十一、成本估算

### 11.1 API 调用成本

| 报告类型 | Token 估算 | 单次成本(Claude Sonnet) |
|---------|-----------|------------------------|
| 周报 | 输入 5000 + 输出 3000 | ~$0.04 |
| 月报 | 输入 8000 + 输出 5000 | ~$0.06 |
| 年报 | 输入 15000 + 输出 8000 | ~$0.12 |

**预估月成本**：假设每周4份周报、每月2份月报、年底1份年报
- 月成本 ≈ $0.04×16 + $0.06×2 ≈ **$0.76**
- 年成本 ≈ $0.76×12 + $0.12 ≈ **$9.24**

成本极低，完全可控。
